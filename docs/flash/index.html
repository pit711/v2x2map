V2X2MAP ITS-G5 Receiver · Web-Flasher

Empfänger direkt im Browser flashen

Verbinde den ESP32-C5 per USB mit deinem Computer und klicke auf Verbinden & Flashen — der Browser erledigt den Rest. Keine Installation, keine Treiber, keine Konfiguration: Der Empfänger leitet seine Kennung automatisch aus der eigenen Hardware-Adresse ab und ist nach dem Neustart sofort per Bluetooth in der V2X2MAP-App sichtbar.

⚠️ Dieser Browser unterstützt kein Web Serial. Bitte Google Chrome oder Microsoft Edge am Computer verwenden — oder den Desktop-Flasher laden. ⚠️ Zugriff auf den seriellen Port wurde blockiert (HTTPS nötig). Firmware v0.2.4
Gerät wird nicht gefunden? Boot-Modus manuell aktivieren: 1. BOOT-Taste gedrückt halten  →  2. RESET kurz drücken (BOOT weiter halten)  →  3. BOOT loslassen und erneut Verbinden & Flashen klicken
Funktioniert mit Chrome, Edge & Opera am Computer (Web-Serial-Technik). Safari, Firefox und Mobilgeräte werden nicht unterstützt — dort bitte den Desktop-Flasher (macOS/Windows) verwenden.

So geht’s

  1. ESP32-C5-Empfänger per USB-Kabel anschließen
  2. Verbinden & Flashen klicken und im Browser-Dialog den seriellen Port wählen (z. B. „USB JTAG/serial debug unit“)
  3. Installieren bestätigen — das Flashen dauert etwa eine Minute
  4. Fertig! Empfänger startet neu → V2X2MAP-App auf dem Handy öffnen
